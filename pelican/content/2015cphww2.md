1.打開命令提示字元

2.cd你要上傳的資料夾

3.git add -A拍快照

4.git commit -m "你更改了哪些項目"

5.git config --global user.email "xxxxxxx@xxxxxxx"使別人知道上傳者是誰

6.git config --global user.name"xxxxxx"使別人知道上傳者是誰

7.git pull origin gh-pages避免衝突及抓取最新資訊

8.git push origin gh-pages上傳囉
